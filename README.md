# Repaso Parcial 1 – Programación en Java
--
## Descripción
Este repositorio contiene una serie de ejercicios desarrollados en **Java**, aplicando el concepto de **modularización mediante métodos**.  
Cada ejercicio está implementado en un método diferente para mantener el código organizado, claro y fácil de entender. 
--
## Tecnologías utilizadas
- Java
- Eclipse IDE
- Git
- GitHub

## Cómo ejecutar el programa

1. Abrir el proyecto en Eclipse IDE
2. Ejecutar la clase Main.java
3. Seleccionar el ejercicio que desea ejecutar
4. Ingresar los datos solicitados en la consola
